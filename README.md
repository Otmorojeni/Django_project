# LEGO STORE


Реализовано API по предметной области LEGO STORE. API создана при помощи Django и СУБД - PostgerSQL. Также реализованы такие эндпоинты как: 

***GET*** - получение данных, таких как: "themes"-тематики наборов, "kits"-наборы лего, "kits/?<params>"-получение группы наборов по фильтру, "brands"-бренды лего наборов, "reviews"-получение отзывов к наборам, "kits/<kit_id>/reviews"-получение отзывов о конкретном наборе.


***POST*** - создание ресурсов, таких как: "themes"-тематика или группы тематик, "kits"-набор или группа наборов, "brands"-бренд или группа брендов, "reviews"-отзыв или группа отзывов.


***PATCH*** - обновление ресурса или группы ресурсов, таких как: "themes", "kits", "brands", "reviews".


***DELETE*** - удаление ресурса или группы ресурсов, таких как: "themes", "kits", "brands", "reviews".


Также реализована автособираемая документация к ендпоинтам API.
